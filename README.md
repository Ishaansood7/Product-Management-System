# Product Management System (Java)

A console-based Product Management System developed using Java and Object-Oriented Programming concepts.

## Features
- Add new products
- View product list
- Update product details
- Delete products
- Exception handling for invalid inputs
- Menu-driven interface

## Technologies Used
- Java
- OOP (Inheritance, Abstraction)
- Exception Handling
- Collections / ArrayList

## Project Structure
User.java – Abstract user class  
Admin.java – Admin functionalities  
Product.java – Product model  
ProductDB.java – Product database operations  
ProjectDemo.java – Main driver program

## How to Run

1. Clone the repository
2. Compile Java files
